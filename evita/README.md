# Evita Project Page

Static GitHub Pages site for:

**Evita: Unified RGB-Event Dense Parsing**

## Publish to GitHub Pages

1. Create a public GitHub repository, for example `evita-page`.
2. Upload everything in this folder to the repository root:
   - `index.html`
   - `.nojekyll`
   - `evita-assets/paper-page1.png`
3. Open the repository on GitHub.
4. Go to **Settings -> Pages**.
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save. The site will appear at:

```text
https://<your-github-username>.github.io/evita-page/
```

For a personal root site, create a repository named:

```text
<your-github-username>.github.io
```

Then upload this folder's files to that repository root. The final URL will be:

```text
https://<your-github-username>.github.io/
```
