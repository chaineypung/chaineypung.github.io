Evita is a unified RGB-Event backbone for robust dense perception in motion blur, low light, adverse weather, and other challenging real-world scenes.

Abstract

Fusing standard RGB frames with asynchronous event streams is a promising route toward robust visual perception in degraded environments. Existing methods often rely on decoupled dual encoders, which increase computational cost, overlook geometric parallax, and suffer from cross-spectral aliasing between dense intensity grids and sparse event dynamics.

Evita introduces a unified RGB-Event backbone for downstream dense prediction. Instead of isolated streams, each encoder layer embeds cross-modal co-learning modules for geometric alignment, frequency-domain texture transfer, and transient global routing. Together with N-ImageNetV2 and stochastic event-representation mixing, Evita learns transferable features that accommodate diverse event formats during finetuning.

Method

Evita targets the physical friction points of RGB-Event fusion: spatial mismatch, spectral heterogeneity, and event-driven long-range context. Its encoder blocks jointly optimize both modalities instead of postponing interaction to a late decoder.

Geometric Parallax Rectification

An adaptive cross-modal deformable operator dynamically aligns event structures with RGB boundaries, correcting spatial offsets from asynchronous sensing.

Harmonic Spectral Resonance

Frequency-domain fusion injects event-driven high-frequency changes while preserving RGB phase structure, reducing noisy spatial aliasing.

Transient Global Routing

Event features act as dynamic queries that route global photometric context from RGB, improving semantic consistency in difficult scenes.

Input

RGB frames and asynchronous event streams enter lightweight modality stems.

Align

Geometric rectification estimates local offsets and samples aligned event features.

Fuse

Spectral resonance combines texture, edges, and transient changes with less noise.

Parse

Unified features are decoded into dense scene understanding outputs.

Experiments

Unified and efficient

A single RGB-Event backbone reduces duplicated computation compared with conventional dual-stream architectures while preserving dense cross-modal interaction.

N-ImageNetV2 pretraining

Strictly aligned RGB-Event pairs and stochastic event-representation mixing help the model generalize across downstream event formats.

Robust to misalignment

Under simulated DDD17 event offsets, Evita-L drops by only about 0.18 mIoU at the extreme shift, demonstrating strong geometric robustness.
